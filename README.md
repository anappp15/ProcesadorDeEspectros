# Procesador de Espectros

Una aplicación web de Streamlit para procesar fotos provenientes de un espectrómetro en espectros (gráficos de intensidad vs. píxel).

## Características
- Subir imágenes espectroscópicas (PNG, JPG, TIFF, etc.)
- Convertir a escala de grises de 8 bits
- Extraer y graficar perfiles de intensidad
- Identificar posiciones e intensidades de los picos

## Trabajo Futuro
- Pestaña de calibración para establecer la escala correcta de número de onda (en lugar de valores de píxel)
